🤖 Robot Control Features

Controls

Feature Description
D-Pad Forward, backward, left, right, stop buttons
Sliders Individual motor control (-100 to 100)
Encoders Read and reset encoder values
Plots Real-time graphs of encoder counts and RPM

Keyboard Shortcuts

Key Action
W / ↑ Forward
S / ↓ Backward
A / ← Turn left
D / → Turn right
Space Stop

WebSocket Commands

Command Response
forward All motors forward at 75%
backward All motors reverse at 75%
left Tank turn left
right Tank turn right
stop Stop all motors
encoders Returns FL, FR, RL, RR counts
reset_encoders Resets all encoder counts
75,75,-50,-50 Custom: FL, FR, RL, RR (-100 to 100)
